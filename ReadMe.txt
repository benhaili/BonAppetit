Une application mobile qui permet de passer des commandes dans un restaurant  et de les enregistrer dans un serveur exeterne,
1- On a utiliser Un serveur Jee avec un api Rest et la framework SpringBoot
2- Une base de donnee Externe  MySql
3- Apres la clique sur  le bouton valider  dans le panier ca va executer une requete post au serveur Jee pour 
enregistrer la commande dans la base de donnee
4-Application mobile 2  qui permer de recuperer les commandes en cours a partir du serveur
